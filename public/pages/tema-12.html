1.2 If else

Consiste en una decisión compuesta; es decir, se evalúa una condición y dependiendo del resultado, en caso de cumplirse, se ejecuta una sentencia o bloque de sentencias, pero en caso de no cumplirse, se ejecuta otra u otras.

La sintaxis del tipo de decisión if else es la siguiente:

Se ilustra a continuación el proceso efectuado cuando se utiliza una instrucción if else.

Con este tipo de instrucción se tiene entonces la ventaja de que se ejecutan acciones, tanto si la condición se cumple, como si no se cumple; es decir que el usuario siempre va a observar un resultado. Sin embargo, esto no es necesario en todos los casos, ya que si por ejemplo una condición se evalúa y se desea que al ser cumplida se ejecute una acción o en caso de no cumplirse no se ejecute, no necesariamente hay un resultado visible al usuario, sino solamente un funcionamiento diferente a nivel interno en el programa.

A continuación, se observan los mismos ejemplos y ejercicios abordados en la instrucción if, pero ya teniendo en cuenta la aplicación del if else.
