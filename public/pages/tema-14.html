1.4 Switch

Esta instrucción de decisión permite efectuar la selección de un bloque de sentencias a ejecutar, dependiendo siempre de un valor analizado; la sintaxis es la siguiente:

Se analiza el valor de una constante (valorAnalizado) y con base en esta se ejecutan las sentencias del caso que corresponda hasta llegar al break.

Las sentencias del bloque default son ejecutadas cuando el valorAnalizado no coincide con ninguno de los valores indicados en los diferentes casos.

Tanto el valorAnalizado como los valores 1, 2, 3… N deben ser de tipo int, char, short o byte.

Se ilustra a continuación el proceso efectuado cuando se utiliza una instrucción switch.
