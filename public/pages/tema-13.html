1.3 If else if

Con este tipo de decisión se abre la posibilidad de evaluar múltiples condiciones y por lo tanto de ejecutar varios grupos diferentes de instrucciones; la sintaxis es la siguiente:

Se ilustra a continuación el proceso efectuado cuando se utiliza una instrucción if else if.

Ejemplos

a. Pedir al usuario que ingrese por teclado el valor que debe cancelar por su plan de telefonía móvil y a través de una instrucción if else if aplique un descuento así:

  • Si el valor a pagar es mayor que 10.000 y menor que 30.000, descuente el 5 % y muestre el nuevo valor a pagar.
  • Si el valor a pagar es mayor que 30.001 y menor que 50.000, descuente el 10 % y muestre el nuevo valor a pagar.
  • Si el valor a pagar es mayor que 50.000 descuente el 15 % y muestre el nuevo valor a pagar.
  • En cualquier otro caso, mostrar un mensaje que indique que el valor ingresado no es correcto.
